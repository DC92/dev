Création d‘une ligne :

Activer (couleur jaune) le bouton L puis :

Cliquer sur la carte et sur chaque point désiré pour dessiner une ligne,

double cliquer pour terminer.

Cliquer sur une extrémité d‘une ligne pour l‘étendre.

Modification d‘une ligne :

Activer (couleur jaune) le bouton M puis :

Cliquer et déplacer un sommet pour modifier une ligne,

Cliquer sur un segment puis déplacer pour créer un sommet,

Alt+cliquer sur un sommet pour le supprimer,

Alt+cliquer sur un segment à supprimer dans une ligne pour la couper,

Ctrl+Alt+cliquer sur un segment de ligne pour le supprimer.
